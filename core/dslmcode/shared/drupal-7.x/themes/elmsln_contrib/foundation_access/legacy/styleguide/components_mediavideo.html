Media Video with Poster

STILL IN DEVELOPMENT. Example with Poster and Play button. This uses a psudo lazy load technique to allow the play button to start the video. To use it, add the video's source url to a data attribute called data-mediavideo-src. A javascript function will is it to populate the src attribute when the button is clicked. Currently works with Youtube and Vimeo.
